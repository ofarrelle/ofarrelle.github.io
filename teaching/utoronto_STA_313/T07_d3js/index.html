Simple D3 Multi-plot Interaction

Click on a bar in the bar chart to highlight countries in that region cluster in the scatter plot below.

View the accompanying main.js file to see how this is accomplished. The Bar and Point objects are saved as variables. D3.js specifies the selection.on() function to add event listeners to each object. When a Bar is clicked, listener object is a tuple of (event, d) where d is the data object for that Bar. From here we can retrieve the cluster of the clicked Bar and it to the function updateSelection(), which updates the opacity of all Bars and Points.

See the documentation at d3js.org for more details. Specifically reference the 2nd paragraph.
